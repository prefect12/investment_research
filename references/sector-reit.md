# REIT 模板

适用于产权类、按揭类或混合型 REIT。

## 核心问题

- 底层资产是什么，租客是谁，租期和续租结构如何
- 资本成本与资产收益率差是否稳定
- 资产组合是否在升级或恶化
- 杠杆和融资节奏是否安全

## 重点指标

- NOI / same-store NOI
- occupancy / lease spread
- AFFO
- leverage / debt maturity
- NAV 与 cap rate

## 管理层重点

- 资产买卖和融资时点是否优秀
- 回购和增发是否逆向思考
- 资产组合重构是否提升长期 NOI 质量

## 常见红旗

- 靠频繁增发维持增长
- AFFO 与真实现金回报背离
- 资产估值过度依赖低利率环境
